Show Me A Thing II


Microcontroller: STM32L476G-DISCOVERY
Sensor Used: MPU-6050 Adafruit 6-DOF IMU

The STM32L476G-DISCOVERY's 4x24 LCD displays X-Axis Acceleration as measured by the MPU-6050. The MPU-6050 uses an I2C interface. The LCD code was adapted from example code provided with the STM32CubeMX L4 expansion. The MPU-6050 code was adapted from Controller Tech's Tutorial on interfacing the MPU-6050 with an STM32. The program also acquires Y-Axis and Z-Axis Acceleration and X,Y,Z-Axis Gyro information. This demonstration only shows the X-Axis Acceleration value for simplicity.
